המדריך המלא לתוכנית הלימודים

תלמידים, הורים ומורים — גלו את כל הנושאים הנלמדים בכל כיתה: חשבון, עברית ועוד, מסודרים בצורה ברורה ונוחה.

תוכנית הלימודים

המסע הלימודי המלא — מכיתה א׳ עד יב׳

כל כיתה מכילה פרק לימודי משלה — עם נושאים, מיומנויות ותכנים המותאמים לגיל ולשלב ההתפתחותי. כאן תוכלו לגלות מה נלמד בכל שנה, מהיסודות ועד לבגרות.

א׳

כיתה א׳

צעדים ראשונים בקריאה וכתיבה בעברית, היכרות עם ספרות וחיבורים פשוטים, ויסודות חשבון — ספירה, חיבור וחיסור.

ב׳

כיתה ב׳

הרחבת אוצר המילים, קריאה שוטפת, הבנת הנקרא, ופיתוח חשבון בסיסי עם כפל וחלוקה ראשונים.

לתוכנית הכיתה ←
ג׳

כיתה ג׳

כתיבה יצירתית, דקדוק עברי בסיסי, לוח הכפל המלא, ומבוא לגיאומטריה ומדידה.

לתוכנית הכיתה ←
ד׳

כיתה ד׳

ניתוח טקסטים ספרותיים, כתיבת חיבורים מובנים, שברים ומספרים עשרוניים, ומבוא למדעי הטבע.

לתוכנית הכיתה ←
ה׳

כיתה ה׳

הרחבת הבנת הנקרא, סוגי כתיבה מגוונים, אחוזים ויחסים, ומבוא לגיאוגרפיה ותנ״ך.

לתוכנית הכיתה ←
ו׳

כיתה ו׳

סיכום היסודי — ביטויים אלגבריים ראשונים, ניתוח ספרותי מעמיק, ומדעי הסביבה והחברה.

לתוכנית הכיתה ←
ז׳

כיתה ז׳

כניסה לחטיבת הביניים — אלגברה, ספרות עברית קלאסית, מדעים ניסויים, ולשון עברית מורחבת.

לתוכנית הכיתה ←
ח׳

כיתה ח׳

גיאומטריה אנליטית, כתיבה אקדמית, היסטוריה ואזרחות, ומדעי הפיזיקה הבסיסיים.

לתוכנית הכיתה ←
ט׳

כיתה ט׳

הכנה לבגרות — פונקציות ומשוואות, ניתוח יצירות ספרות, כימיה ובيולוגיה, ואנגלית מתקדמת.

לתוכנית הכיתה ←
י׳

כיתה י׳

מסלולי בגרות מתחילים — מתמטיקה ברמות שונות, עברית ספרותית, מדעים ומקצועות בחירה.

לתוכנית הכיתה ←
י״א

כיתה י״א

הכנה מעמיקה לבחינות בגרות, בחירת מגמות, פרויקטים עצמאיים, ופיתוח חשיבה ביקורתית.

לתוכנית הכיתה ←
י״ב

כיתה י״ב

שנת הבגרות — בחינות מסכמות, עבודות גמר, ומעבר לחינוך הגבוה. שיא המסע הלימודי.

לתוכנית הכיתה ←
תוכנית הלימודים

המקצועות שמלווים אתכם מכיתה א׳ עד יב׳

האתר מציג את נושאי הלימוד המרכזיים בכל שכבת גיל — ממתמטיקה ועברית ועד הבנת הנקרא וכתיבה יצירתית. כאן תוכלו לראות כיצד רמת המורכבות גדלה בהדרגה לאורך שנות הלמידה, ולהתמצא בתוכנית הלימודים לכיתות א׳ עד יב׳ בצורה ברורה ונגישה.

  • מתמטיקה

    מספרים ופעולות חשבון בכיתות הנמוכות, אלגברה, גיאומטריה וחשבון דיפרנציאלי בכיתות הגבוהות.

  • שפת האם — עברית

    אלפבית וקריאה ראשונית, הרחבת אוצר מילים, תחביר, ניתוח טקסטים ספרותיים ועיוניים.

  • הבנת הנקרא

    פיתוח אסטרטגיות קריאה, הסקת מסקנות, ניתוח רעיון מרכזי ועיבוד מידע מורכב.

  • כתיבה יצירתית ועיונית

    כתיבת סיפורים קצרים, חיבורים, טיעון מנומק, וכתיבה אקדמית בכיתות הבוגרות.

  • דקדוק ולשון

    חלקי דיבר, זמני פועל, מבנה משפט, ניקוד ואיות — מיסודות הכתיב ועד ניתוח לשוני מעמיק.

  • יסודות הלמידה

    מיומנויות ארגון, למידה עצמאית, עבודת צוות ופיתוח חשיבה ביקורתית לאורך כל שלבי הלימוד.

למי האתר מיועד?

תוכנית הלימודים מכיתה א עד יב — עבור כל מי שמלווה את מסע הלמידה

תלמידים

רוצים לדעת מה לומדים בכיתה שלכם? כאן תוכלו לגלות את כל הנושאים הצפויים בכל שכבת גיל — ולהגיע מוכנים לכל שיעור.

הורים

עקבו אחר ההתקדמות הלימודית של ילדיכם לאורך כל שנות הלמידה. הבינו מה נלמד בכל כיתה ואיך לתמוך בצמיחה שלהם בכל שלב.

מורים

השתמשו באתר כמדריך מובנה לתכנון שיעורים ולתקשורת עם הורים ותלמידים. מסמך עזר ברור לכל מקצוע ולכל שכבת גיל בחינוך.

גלו את תוכנית הלימודים לפי כיתה

האתר שלנו הוא מדריך עזר חינוכי ברור ומסודר לכל שלבי בית הספר — מכיתה א׳ ועד יב׳. בין אם אתם הורים, מורים או תלמידים, תמצאו כאן את נושאי הלמידה המרכזיים בכל מקצוע ובכל שכבת גיל.
